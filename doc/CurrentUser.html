current_user() click to toggle source

Get the current user: 

 - if it is already set, then return it without doing a lookup
 - otherwise, find the user via current_user_find(rubygem_current_user_id)

If the current user is not set, and the current user id is nil, then this returns nil because current_user_find returns nil. 

    # File lib/sixarm_ruby_current_user.rb, line 34
34:  def current_user
35:   @current_user ||= current_user_find(current_user_id) 
36:  end
